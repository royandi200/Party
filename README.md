# Party 🛥️

**App móvil de registro para el evento AIRA Party — Yate Majestic**

Visualiza en tiempo real quién está registrado en cada puesto del yate, con distinción visual por género. Mismo estilo dark que AIRA: azul `#004fff` + lima `#e1fe52`.

## Features

- 🗺️ **Mapa del yate interactivo** — 3 cubiertas (VIP, General, Social) con 60 puestos
- 👥 **Distinción por género** — azul para hombres, lima para mujeres, dorado para VIP
- 📊 **Stats en tiempo real** — ocupación, balance de género, cupos disponibles
- 📋 **Lista de asistentes** — con nombre, zona, puesto e Instagram
- ✍️ **Registro rápido** — formulario mobile-first ultra simple
- ⭐ **Pass VIP** — toggle para acceso especial

## Stack

- HTML + CSS + JS puro (sin frameworks)
- Fuente: Figtree (Google Fonts)
- Colores AIRA: `#004fff` azul · `#e1fe52` lima · `#08101f` fondo dark

## Uso

Abre `party-app.html` directamente en el navegador.
